# Barbershop Booking

Одностраничное приложение для онлайн-записи в барбершоп: витрина услуг/барберов, личный кабинет с управлением визитами, перенос/отмена, профайл пользователя и мультиязычность (ru/uk/en).

## Features
- Запись на стрижку с выбором барбера, услуги, даты и времени (динамические свободные слоты)
- Личный кабинет: будущие/прошлые визиты, отмена, перенос, авто‑отметка no-show
- Профиль пользователя (имя/телефон/email), подтяжка данных в форму записи
- Барберы с фото (загрузка через админку)
- i18n (ru/uk/en) + переключатель языка
- Honeypot + rate-limit (3 запроса / 10 минут) на создание записи
- Статусы брони, сообщения об успехе/ошибках, стилизованные уведомления

## Стек
- Python 3.13, Django 6
- SQLite (по умолчанию)
- HTML/CSS/JS без фронт-фреймворков

## Установка и запуск
```bash
git clone <repo>
cd BarbershopPP/barbershop
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp ../.env.example .env  # заполните ключ и нужные настройки
python manage.py migrate
python manage.py runserver
```

## Тесты
```bash
python manage.py test
```
