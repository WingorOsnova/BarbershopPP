# BarbershopPP
Проект: Онлайн-система записи для барбершопа Цель: Чтобы клиенты могли онлайн записываться к барберам, а админ — управлять расписанием и заказами. Стек (по желанию клиента): Python + Django (или Flask/FastAPI, но лучше Django), HTML/CSS, немного JS. Можно Tailwind. Я хочу простую, но “живую” систему, не тупо статический сайт.
